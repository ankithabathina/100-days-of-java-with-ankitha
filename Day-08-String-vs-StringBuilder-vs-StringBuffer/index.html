📘 Day 8 - String vs StringBuilder vs StringBuffer

100 Days of Java with Ankitha

🎯 Objective

Learn the differences between String, StringBuilder, and StringBuffer, including mutability, thread safety, performance, and internal working.

📖 What is String?

String is an immutable class in Java. Once a String object is created, its value cannot be changed. Any modification creates a new object.

📖 What is StringBuilder?

StringBuilder is a mutable class that allows modifying the same object. It is faster because it is not synchronized.

📖 What is StringBuffer?

StringBuffer is also mutable but thread-safe because its methods are synchronized. It is mainly used in multi-threaded applications.
