📖 What is ArrayList?

ArrayList is a resizable array implementation of the List interface. It stores elements dynamically and automatically increases its capacity whenever the existing array becomes full.

⭐ Features

  • Resizable (Dynamic Size)
  • Maintains Insertion Order
  • Allows Duplicate Elements
  • Allows Null Values
  • Fast Random Access using Index

📚 Important Methods

Method Description
add() Adds an element.
get() Returns an element.
set() Updates an element.
remove() Removes an element.
contains() Checks whether an element exists.
size() Returns the number of elements.

⏱️ Time Complexity

Operation Complexity
add() O(1)

remove() O(n)
