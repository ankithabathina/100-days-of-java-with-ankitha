📖 What is Java 8?

Java 8 is one of the most important releases of Java. It introduced Functional Programming, Lambda Expressions, Stream API, Method References, Optional Class, and the new Date & Time API. These features make Java applications more concise, maintainable, and efficient.

⭐ Major Features of Java 8

Feature Description
Lambda Expression Anonymous function used to reduce code.
Functional Interface Contains exactly one abstract method.
Method Reference Shortcut for Lambda Expressions.
Default Method Method with implementation inside an interface.
Static Method Static method inside an interface.
Stream API Processes collections efficiently.
Optional Class Avoids NullPointerException.
Date & Time API Improved date and time handling.

🔹 Lambda Expressions

A Lambda Expression is an anonymous function introduced in Java 8. It is mainly used to implement Functional Interfaces without creating separate implementation classes. Lambda expressions reduce boilerplate code and improve readability. 


Syntax

(parameters) -> expression

Example

() -> System.out.println("Hello Java");

(x, y) -> x + y

Advantages

  • Less Code
  • Better Readability
  • Functional Programming
  • Easy Event Handling
  • Works Well with Stream API

🔹 Functional Interface

A Functional Interface is an interface that contains exactly one abstract method. It can have multiple default and static methods, but only one abstract method. The @FunctionalInterface annotation ensures that only one abstract method exists. 


@FunctionalInterface

interface Greeting{

    void sayHello();

}

Examples

  • Runnable
  • Callable
  • Comparator
  • Consumer
  • Supplier
  • Predicate
  • Function

🔹 Method Reference

Method Reference is a shorthand notation for Lambda Expressions. Instead of writing a Lambda Expression, we can directly refer to an existing method. This improves code readability. 


System.out::println

ClassName::staticMethod

object::instanceMethod

ClassName::new
